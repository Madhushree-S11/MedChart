# MedChart
A website to assist doctors and nurses to monitor vitals of a patient according to csv reports virtually

the CSV file must be in the format of:

readingTimestamp - ISO 8601 formatted timestamps 

heartRate - Heart rate in beats per minute 

bloodPressureSystolic - Systolic pressure 

bloodPressureDiastolic - Diastolic pressure 

temperature - Body temperature in Celcius 

Already a sample input report has been uploaded and tested.
